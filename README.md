# 仮想環境
## 仮想環境の作成
* このディレクトリで、`$ python3 -m venv venv`
* 後ろの `venv` は、envname と言って仮想環境の名前
* .gitigonore と連動させておく
## activate
* `$ . venv/bin/activate`
## deactivate
* `([envname])$ deactivate`
